# Gap 分析：现有空白 → 新 Benchmark 定位

## 一、现有 Benchmark 的系统性盲区

### 1. 序列-语言联合理解（核心空白）

现有benchmark几乎全部是**纯序列**（ProteinGym, CASP）或**纯文本**（MedQA, BLURB），缺少：

| 空白 | 描述 | 难度估计 |
|------|------|---------|
| 分子序列→自然语言机制解释 | 给 `<mol>SMILES</mol>`，解释其药效团、MoA | ★★★★ |
| 蛋白序列→功能问答 | 给 `<protein>seq</protein>`，回答其结构域、底物、抑制剂 | ★★★★ |
| RNA序列→调控功能 | 给 `<rna>seq</rna>`，预测其靶基因或二级结构功能 | ★★★★★ |
| 文本→正确序列标记 | 给疾病描述，输出相关靶标的 `<protein>` 序列 | ★★★★★ |
| 多序列类型混合推理 | 同时包含 mol/protein/rna 的跨域推理题 | ★★★★★ |

**这是本项目独特定位：没有任何现有benchmark覆盖序列标记的理解与生成能力。**

---

### 2. 跨域因果推理

| 空白 | 描述 | 对应本项目数据集 |
|------|------|----------------|
| 变异→表型→疾病 | 从基因变异推断疾病表现机制 | ClinVar + HPO + gene_summary |
| 药物→靶标→通路→疾病 | 从化合物SMILES推断MoA到疾病适应症 | chembl_protein_merged + drugbank |
| 序列变异→结构变化→功能改变 | 从点突变预测蛋白构象变化和功能影响 | uniprot_swiss + trembl |
| 基因表达→细胞状态→组织表型 | 从差异表达推断细胞命运 | gene_seq_refseq + pubmed |

---

### 3. 知识深度层次（当前benchmark缺乏分层）

现有benchmark通常只测一个层次，缺少**能力边界的阶梯性设计**：

```
Level 1: 事实记忆  —— "EGFR的全称是什么？"
Level 2: 概念理解  —— "EGFR激活后发生什么？"
Level 3: 机制推理  —— "T790M突变如何导致吉非替尼耐药？"
Level 4: 跨域整合  —— "设计克服T790M耐药的第三代EGFR抑制剂策略"
Level 5: 序列操作  —— "给出EGFR T790M突变体序列，预测奥希替尼结合位点"
```

本项目新benchmark应覆盖 Level 3–5，对应6层实体框架。

---

### 4. 分子生物学六大已知空白汇总

| 领域 | 现有最高水平 | 空白 |
|------|------------|------|
| 蛋白质结构 | CASP15，复合体预测 | 动态构象、本征无序区 |
| 蛋白质功能 | ProteinGym 217 assays | 多蛋白体系、酶催化定量 |
| PPI | AUROC ~89%（有泄露） | 条件依赖性互作、弱结合 |
| 变异效应 | ClinVar AUC 0.73 | 罕见变异、非编码区机制 |
| 基因表达 | Pearson r 0.875（Enformer） | 单细胞扰动（基线都打不过） |
| 药物-靶标 | BindingDB AUROC 0.99 | 变构位点、共价结合 |

## 二、新 Benchmark 设计原则

### 维度一：按本项目6层实体分类

每一层实体设计专属能力测试，确保全覆盖：

1. **分子实体层**：药物/蛋白质/RNA/代谢物的序列理解与跨实体映射
2. **机制过程层**：信号通路/MoA/PTM/耐药机制推理
3. **结构功能域层**：Pfam/活性位点/折叠类型的识别与功能推断
4. **细胞组织层**：细胞类型/组织特异性表达/发育阶段理解
5. **变异表型层**：ClinVar变异→HPO表型→疾病的因果链推理
6. **方法技术层**：实验设计理解、组学技术原理、计算工具选择

### 维度二：按能力类型

| 能力类型 | 示例任务 | 对应难度 |
|---------|---------|---------|
| **序列识别** | 识别给定序列的生物类型（DNA/RNA/蛋白/化学物质） | L1 |
| **功能注释** | 从序列预测功能/定位/修饰 | L2 |
| **机制推理** | 从突变推断功能改变 | L3 |
| **跨域整合** | 药物+蛋白+疾病三角推理 | L4 |
| **生成验证** | 生成正确的序列标记 + 机制描述 | L5 |

### 维度三：现有benchmark未覆盖的独特测试点

- [ ] 中英文混合生命科学推理（中文临床+英文基础研究）
- [ ] 序列标记（`<mol>`, `<protein>`, `<rna>`）的正确使用与解读
- [ ] 跨数据库知识整合（ChEMBL ID → UniProt → HPO → 疾病）
- [ ] 文献引用准确性（给定摘要，模型引用正确还是幻觉）
- [ ] 实验可行性判断（给定实验方案，识别设计缺陷）

## 二-B、AI综述补充的新维度（与本项目高度相关）

### 单细胞与空间组学（之前遗漏）

| 领域 | 现状 | 核心空白 |
|------|------|---------|
| scRNA-seq扰动预测 | 所有FM均输给PCA基线（Nature Methods 2025） | **因果机制建模**缺失；没有模型真正理解基因调控逻辑 |
| 空间基因表达预测 | 11种方法对比（Nat. Commun. 2025）；Pearson r~0.7–0.8 | 跨组织/跨物种泛化；高基因数覆盖 |
| 单细胞注释 | scSuperAnnotator提供方法对比平台 | 稀有细胞类型；新兴细胞状态 |

### 智能体评测（2025–2026新兴，本项目可切入）

| 领域 | 现状 | 核心空白 |
|------|------|---------|
| scRNA-seq分析 | scBench (394题) | 中文环境；与本地数据库联动 |
| 临床EHR任务 | MedAgentBench | 中文临床EHR；多步推理 |
| 生物安全边界 | BioAgent Bench | 边界定义不统一 |
| **跨领域序列操作** | **无** | **全空白：`<mol>/<protein>/<rna>`标记操作** |
